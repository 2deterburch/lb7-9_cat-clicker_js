# Cat Treat Clicker

Інтерактивна браузерна гра, створена на React + Vite.

Гравець клікає на котика, збирає ласощі, купує апгрейди та відкриває нові скіни котиків.

---

## Демо

 https://2deterburch.github.io/lb7-9_cat-clicker_js/

---

## Основний функціонал

-  Клік по котику (основний заробіток)
-  Накопичення ласощів
-  Апгрейди:
  - збільшення кліку
  - пасивний дохід
  - множник
- Котяча коробка (рандомні ефекти)
-  Скіни котиків (відкриття і вибір)
-  Престиж система (CatCoins)
-  Досягнення
-  Автозбереження прогресу (localStorage)
-  Звук кліку (мяу)
-  Анімації та сучасний дизайн

---

##  Технології

- React (useState, useEffect)
- Vite
- JavaScript (ES6+)
- CSS (Flexbox, анімації)
- GitHub Pages (деплой)

---

## Що демонструє проєкт

- Роботу з компонентами React
- Використання props
- Керування станом через useState
- Побічні ефекти через useEffect
- Роботу з localStorage
- Побудову інтерактивного UI
